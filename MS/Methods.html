METHODS for ‘same data different analysts’ project, ‘Many EcoEvo Analysts’ dataset

Luke J. Eberhart-Phillips and Martin Bulla

2020-08-12 14:09:52

General background

Upon reading the data description and metadata for ‘Many EcoEvo Analysts’ dataset provided by the project coordinators of ‘same data different analysts’ project we have A PRIORI decided upon the procedures and analyses to answer the key question: “To what extent is the growth of nestling blue tits (Cyanistes caeruleus) influenced by competition with siblings?”.

We have anticipated, but not expected, that the actual data differ from the data description. Although the nests with reduced and increased brood size are well matched by hatch date (Figure 1), in 20% they differ by more than one chick in hatched brood size and in 9% of case the differences are extreme (i.e., brood size differences of 3 or more chicks).

Importantly, the data description indicated that in broods with 4-5 chicks only one chick was taken or added and in broods with 12-16 chicks four chicks were taken or added. This is not the case. Broods of different than indicated size were used and other chick numbers were added and taken (Figure 2). Also, we have not expected that a key variable number of chicks at the start of experiment (‘rear_Cs_at_start_of_rearing’) will be missing 90% of the cases. Thus, we do not use this variable in chick size models (see below).

Model specification

We have two main sets of the models, as specified in our a priori protocol and additional exploratory models as a part of extended material.

(1) Chick mortality model

We specified the change in chick number in the rearing nests (change_chick_n) as a difference between the number of chicks at the start of the experiment (rear_Cs_at_start_of_rearing) and number of chicks on day 14 of the experiment (d14_rear_nest_brood_size) and fitted this variable as response. The number of chicks to enlarge or reduce the brood (net_rearing_manipulation; range -4 to 4, median 0; note that zero indicates the control nests) and brood sex ratio (brood_sex_ratio; range 0-1, median 0.5) as explanatory variables. To control for pseudoreplication (multiple-sampling), we specified the measurer (day14_measurer), the compartment of the study area (rear_area), the year (hatch_year) and the day on which the rear nest hatched left centered within year, i.e. minimum hatch day within each year subtracted from hatch day of a given nest (rear_nest_OH_l) as random intercepts. In total, we had 357 nests for which these data were available.

Model M - chick mortality

    lmer(change_chick_n ~ 
         net_rearing_manipulation  +       
         brood_sex_ratio +     
         (1|day14_measurer) + 
         (1|rear_area) +      
         (1|rear_nest_OH_l) + 
         (1|hatch_year),       
         REML = TRUE        
         )

(2) Chick size models

The dataset contained a chick morphometrics measurements from only one time point (i.e., day 14): tarsus length (day_14_tarsus_length) and body mass (day_14_weight). Thus, we could not have estimated the actual growth rate given the treatment. We thus infer the growth differences based on treatment only. In addition to tarsus length and body mass, we calculated a composite variable of body mass index (body_mass_index):

\[day\_14\_body\_mass\_index = day\_14\_weight / (day\_14\_tarsus\_length^2)\]

In humans, the body mass index broadly categorizes a person as underweight, normal weight, overweight, or obese based on tissue mass (muscle, fat, and bone) and height (structural size). In our case we use tarsus instead of height and we can then derive a value that classifies a given chick as underweight for its tarsus size (e.g., maybe due to intense sibling competition), normal, or overweight (e.g., maybe due to relaxed sibling competition).

We then specified three subsets of models, each with one of the chick morphometric variables as a response. The number of chicks to enlarge or reduce the brood (net_rearing_manipulation; range -4 - 4, median 0; note that zero indicates the control nests) was specified as an explanatory variable, testing for the treatment effect. We controlled the model for chick sex (chick_sex_molec; m - male, f - female, u - unknown), and brood sex ratio (brood_sex_ratio; range 0-1, median 0.5, calculated as described in Chick Mortality Model) because female and male blue tit chicks grow differently (Mainwaring, Dickens, and Hartley 2010). Brood sex ratio was calculated as proportion of males in the nest (1 = all males, 0 = all females), while indicating unknown sex of chicks as 0.5. To control for pseudoreplication (multiple-sampling), we specifying the measurer (day14_measurer), the compartment of the study are (rear_area), the year (hatch_year), the day on which the rear nest hatched (left centered) within year, i.e. minimum hatch day within each year subtracted from year hatch day of that given year (rear_nest_OH_l), the identification of rearing nest (rear_nest_breed_ID) and of genetic parents (hatch_mom_Ring and genetic_dad) as random intercepts. Such data were available for 2550 nests.

Model Ta - tarsus

    lmer(day_14_tarsus_length ~ 
         net_rearing_manipulation + chick_sex_molec + 
         brood_sex_ratio +
         (1|day14_measurer) + 
         (1|rear_area) + 
         (1|rear_nest_OH_l) + 
         (1|hatch_year)  +
         (1|rear_nest_breed_ID) +
         (1|hatch_mom_Ring) + 
         (1|genetic_dad),
         REML = TRUE
         )

Model Wa - weight

    lmer(day_14_weight ~ 
         net_rearing_manipulation + chick_sex_molec + 
         brood_sex_ratio +
         (1|day14_measurer) + 
         (1|rear_area) + 
         (1|rear_nest_OH_l) + 
         (1|hatch_year)  +
         (1|rear_nest_breed_ID) +
         (1|hatch_mom_Ring) + 
         (1|genetic_dad),
         REML = TRUE
         )

Model Ba - body mass index

    lmer(body_mass_index 
         net_rearing_manipulation + chick_sex_molec + 
         brood_sex_ratio +
         (1|day14_measurer) + 
         (1|rear_area) + 
         (1|rear_nest_OH_l) +

Note that the above described models (Ta, Wa, Ba) differ from those proposed in the a priori protocol because we followed our a priori decisions about analyses. There were not enough data for the number of chicks at the start of the experiment (rear_Cs_at_start_of_rearing) and hence we have not used it in the models. The correlation between net_rearing_manipulation and d14_rear_nest_brood_size was 0.68, i.e. higher than our predefined 0.6 threshold. Thus, according to the a priori protocol we used only one variable. As net change in chick number captures the actual experimental manipulation, we use this variable through out. Note however, that we have also fitted models with d14_rear_nest_brood_size instead of net_rearing_manipulation (named models Tb, Wb, Bb), which generated same results (see models a and b in Table T, Table W, and Table B in the results).

To verify whether the effects of the treatment differs between females and males, we modified the above models by adding an interaction between sex and net change in chick numbers net_rearing_manipulation (as indicated in our a priori protocol]). See models c in Table T, Table W, and Table B in the results.

In addition, and as intended, we have also fitted the a priori defined models that control for experimental pairs (i.e., reduced, enlarged). For this paired analyses we have thus used only the pairs that differ in the original brood size by no more than one chick (75% of nest pairs, N = 303; see figure 2 above). As not all data in the dataset had pairs assigned and as this method does not work for control nests, we have refitted the Table T, W and B models to the reduced data and included random slope pair_ID. These results ( Tables TEP, TWE, TBE) are similar to those from the initial models (Tables T, W, B in Results).

(3) Exploratory models

To explore whether the results are consistent even if we specify the models according to our a priori protocol (but regardless of our a priori assumptions), we added to the Chick Size Models (models T, W, B) an interaction between d14_rear_nest_brood_size and net_rearing_manipulation, and specified models without an interaction but with both variables present. To faciliate interpretation of the interactions and also to compare the standardised effect sizes between variaous responses and models, we present these exploratory results together with the above models (T, W, B), but with the response and fixed effects scaled (i.e., Table TE, Table WE and Table BE). The results of these exploratory models were fundamentally similar to our simple models (presented in Tables T, W, B Results).

Statistical analyses

We used R version 4.0.2 (R-Core-Team 2020) for all statistical analyses and the ‘lme4’ package (Bates et al. 2015) for fitting the mixed-effect models using restricted maximum likelihood (for AIC comparison using maximum likelihood). We used the sim function from the arm package and a non-informative prior-distribution (Gelman and Hill 2007; Gelman and Su 2018) to create a sample of 5,000 simulated values for each model parameter (i.e. posterior distribution). We report effect sizes and model predictions by the medians, and the uncertainty of the estimates and predictions by the Bayesian 95% credible intervals represented by the 2.5 and 97.5 percentiles (95% CI) from the posterior distribution of 5,000 simulated or predicted values. All results are reproducible with the open access data and code available together with all materials on our GitHub repository (temporally hosted on Dropbox).

In all model comparisons we assessed the model fit by Akaike’s Information Criterion (Anderson 2008) generated by the ‘AIC’ function in R.

In all models the model assumptions were met, except for Table M model where error distribution was not normal as the chick mortality data are bounded at zero. However, the Gaussian models are robust against violation of such assumptions and can be fitted to the binomial and Poisson distributions (Knief and Forstmeier 2018).
